# Pattern Playground

An interactive browser-based tool for exploring repeating pattern methods from Paul Jackson's book [*How to Make Repeating Patterns*](https://www.laurenceking.com/products/how-to-make-repeating-patterns).

Select a font and letter, choose a symmetry operation, adjust its settings, and see the result rendered live. Toggle between a single tile view and a tiled preview to see how the pattern repeats. Export your designs as SVG or PNG.

Built with Svelte + Vite. All rendering happens in the browser.
